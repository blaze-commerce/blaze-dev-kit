# Getting Started

Welcome to the Blaze Dev Kit. This section walks you through everything you need to deploy the kit to a Kinsta-hosted WordPress site.

## Steps

1. **[Requirements](requirements.md)** - What you need before starting
2. **[Installation](installation.md)** - Download, upload, extract, clean up
3. **[Initialization](initialization.md)** - Run `/init` to configure Claude Code for your site

## Quick Overview

The Blaze Dev Kit is a `.claude/` directory that you deploy to the root of your WordPress site on Kinsta. After extraction, you run `/init` inside Claude Code, which generates a site-specific `CLAUDE.md` file with all your site's details, available commands, and safety rules.

From that point on, Claude Code has full knowledge of your site and can perform maintenance, optimization, development, and auditing tasks safely.
