# Services

The Blaze Dev Kit supports four core service areas. Each service has dedicated commands, skills, and templates.

| Service | Commands | Primary Use |
|---------|----------|-------------|
| [WordPress Maintenance](wpm.md) | `/wpm` | Core, plugin, and theme updates |
| [Site Optimization](optimization.md) | `/pagespeed`, `/accessibility`, `/bestpractices` | Lighthouse score optimization |
| [Web Development](development.md) | Skills + Agents | Custom WordPress development |
| [SEO & Marketing](seo-marketing.md) | Skills | Technical SEO and performance |

Each service has a corresponding base template in `.claude/templates/` that gets merged into `CLAUDE.md` during `/init`.
