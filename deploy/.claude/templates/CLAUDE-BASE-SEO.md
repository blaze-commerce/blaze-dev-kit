# SEO & Marketing - Service Base

## Technical SEO Guidelines

### Rank Math (Standard Stack)
- Rank Math is the default SEO plugin for Blaze Commerce sites
- Configure focus keywords per page/product
- Enable schema markup for products, articles, local business
- XML sitemap auto-generated by Rank Math

### On-Page SEO Checklist
- [ ] Title tag: Under 60 characters, includes primary keyword
- [ ] Meta description: Under 160 characters, includes call-to-action
- [ ] H1: One per page, includes primary keyword
- [ ] Heading hierarchy: H1 > H2 > H3 (no skipping levels)
- [ ] Image alt text: Descriptive, includes keywords where natural
- [ ] Internal links: At least 2-3 per page
- [ ] Canonical URL: Set correctly (especially for product variations)

### Schema Markup
- **Products**: Use WooCommerce + Rank Math auto-generated schema
- **Articles/Blog**: Use Rank Math Article schema
- **Local Business**: Configure in Rank Math Local SEO module
- **FAQ**: Use Rank Math FAQ block or schema

### Performance = SEO
Core Web Vitals directly affect search rankings:
- LCP < 2.5s (Good)
- FID/INP < 200ms (Good)
- CLS < 0.1 (Good)

Use `/pagespeed` to ensure these targets are met.

### Image SEO
- Descriptive file names (not IMG_001.jpg)
- Alt text on all images
- Proper dimensions (width/height attributes)
- AVIF/WebP format via ShortPixel
- Lazy loading for below-fold images

### Robots.txt
Located at site root. Do not modify without approval. Standard rules:
```
User-agent: *
Disallow: /wp-admin/
Allow: /wp-admin/admin-ajax.php
Sitemap: {SITE_URL}/sitemap_index.xml
```
